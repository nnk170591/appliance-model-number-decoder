# Whirlpool Brand Prefix Guide: KitchenAid, Maytag, Jenn-Air & More

Map any WPL-family model number prefix to its brand. Derived from analysis of 44,783 models.

## Quick Rules

| First Letter | Brand | Confidence |
|-------------|-------|------------|
| K | **KitchenAid** | 100% — all 8,867 K-prefix models |
| J | **Jenn-Air** | ~95% — some J-prefixes are Maytag (see below) |
| T | **Estate** | High — T-prefix models |
| I | **IKEA** | High — I-prefix models |
| W | **Whirlpool** | High — genuine Whirlpool |
| M | **Maytag** | High — M-prefix models |
| ###.##### | **Kenmore** | 100% — digit-dot-digit pattern |

## Kenmore
Kenmore models use a digit-dot-digit pattern: `106.xxxxx`, `110.xxxxx`, `665.xxxxx`, `116.xxxxx`, `596.xxxxx`

## Whirlpool (genuine)
W-prefix (WRF, WFE, WDT, WMH, WDF, WFG, WED, WGD, WRT, WTW, WOS, WRS, WFW, WOD, WOC...), plus: DU0, DU1, DU8, DU9, GU, GS, GB, GH, GG, GE, GM, GR, GD, GJ, GL, GT, GC, GX, SF, SC, RF, ED, EV, EH, ET, IR, RH2-5, RB...

## Split Prefixes (Tricky)

| Prefix | Brand | Rule |
|--------|-------|------|
| DU0, DU1, DU8, DU9 | Whirlpool | DU + 0/1/8/9 |
| DU2, DU3, DU4, DU5, DU6, DU7 | Maytag | DU + 2-7 |
| RB, RH | Whirlpool | R + B or H |
| RT, RU, RG, RS, RE, RA | Roper | Most other R-prefixes |

## Y-Prefix (Variants)
Y-prefix indicates a variant/Canadian model. The letter **after** Y determines the brand:
- YW = Whirlpool
- YM = Maytag  
- YA = Amana
- YK = KitchenAid
